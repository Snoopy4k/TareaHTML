"Que relacion hay entre los Elementos y las Etiquetas?" Noticia - "<"h3">"

Parrafo: "<"p">"

Los elementos dentro de un archivo HTML se distinguen mediante etiquetas. Cada elemento tiene una etiqueta asociada.

Por ejemplo, todo lo que escribamos dentro de una etiqueta "<"p">" se transformará en texto plano dentro de nuestra página web.

"Cuantos tipos Listas hay?" Noticia 2 - "<"h3">"

Etiqueta "OL": "<"p">"

Los Order List "OL", estas etiquetas hacen que si le sumo la Etiqueta "li" dentro de la estructura "ol", una onda así, ejemplo:

(Dentrode de mi VSC) Ahora si le sumo la Etiqueta "li" dentro de la estructura "ol", una onda así

"<"ol"">

"<"li"">" Información "<"/li"">"

"<"li"">" Oracion "<"/li"">"

"<"li"">" Palabra "<"/li"">"

"<"/ol">"

"

En mi pagina web se veria así, se ve ordenada secuencialmente:

  1. Informacion
  2. Oracion
  3. Palabra
  4. Etcetera

"Etiquetas de Formato"

"Las más usadas en Textos"

  • "<" h1 > a "<" h6 > - Encabezados
  • "<" p >" - Párrafos
  • "<" br >" - Salto de linea o "punto aparte"
  • "<" b >" - Letra en Negrita
  • "<" hr / > - Linea de separación
  • "<" i >" - Letra en Cursiva
  • "<" Strong >" - Letra en Negrita mas fuerte
  • "<" em >" - Letra en Cusiva con enfásis
  • "<" u >" - Letra Subrayado
  • "<" s >" - Letra tachada
  • "<" a >" - Crea un hipervinculo hacia otra pagina web, un enlace

"Las más usadas en Estructura del HTML"

  • "<" header> - Cabecera de una sección o página
  • "<" title >" - El nombre del titulo del documento, es el nombre que va figurar con la barra de titulo
  • "<" body> - Cuerpo de la página
  • "<" img sc="Ruta de imagen, lugar de donde viene la imagen HTTP://.... "> - Incrusta una imagen
  • "<" div> - Es un contenedor AGRUPA ELEMENTOS
  • "<" span> - Titulos o Solapas que se alinean uno al lado de otro
  • "<" article> - Etiqueta de bloque de pagina
  • "<" section> - Etiqueta de bloque de pagina
  • "<" main> - Contenido principal de la pagina
  • "<" footer> - Debajo del contenido principal, generalmente son enlaces a redes sociales
